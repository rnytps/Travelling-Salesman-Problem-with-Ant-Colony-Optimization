# Travelling-Salesman-Problem-with-Ant-Colony-Optimization
ini adalah tugas dari mata kuliah kecerdasan buatan dengan menggunakan ant colony optimization

di folder ini ada 4 file yaitu
1. acoloop yang menentukan titik koordinatnya menggunakan jarak antar titik atau graph dan melakukan looping atau melewati titik yang sama secara berulang
2. ant sama seperti acoloop tetapi tidak melakukan looping jadi tidak melewati titik yang sama secara berulang
3. matriksloop menentukan titik nya menggunakan matriks dan melakukan looping
4. matriks sama seperti matriksloop tetapi tidak melakukan looping
